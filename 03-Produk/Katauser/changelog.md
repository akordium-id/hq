## ğŸ“ Key Decisions Log

### 2026-01-06: Product Concept Validation

**Decision:** Proceed dengan validation phase sebelum full development

**Rationale:**
- Lean startup methodology: validate before build
- Market research needed untuk confirm problem-solution fit
- Customer interviews akan provide critical insights
- Landing page quick win untuk test market interest

**Outcome:**
- Created validation plan (M1-M4 milestones)
- Defined Go/No-Go decision criteria
- Set up documentation structure

---

### 2026-01-06: Tech Stack Selection

**Decision:** Use Go + Tanstack Start + PostgreSQL

**Rationale:**
- **Go:** Performance, concurrency, company expertise (Najib)
- **Tanstack Start:** Modern React framework, SSR/SSG, better DX than Next.js
- **PostgreSQL:** Company standard, robust features, Oracle Cloud support

**Alternatives Considered:**
- Backend: Node.js (rejected untuk performance), Python (rejected untuk scalability)
- Frontend: Next.js (rejected - prefer Tanstack Start), Laravel (rejected - Go company focus)
- Database: MySQL (rejected - migrating to PostgreSQL company-wide)

**Outcome:**
- Tech stack locked for MVP
- Architecture document created
- Development environment planned

---

### 2026-01-06: MVP Feature Scope

**Decision:** Limit MVP to 2 core features (Public Board + Roadmap)

**Rationale:**
- Time-to-market priority (launch in 5 months)
- Focus on differentiation (Bahasa Indonesia + Telegram)
- Avoid feature creep
- Validate core value proposition first

**Features Excluded from MVP:**
- AI duplicate detection (post-MVP)
- Changelog generator (post-MVP)
- Private boards (premium feature)
- Advanced analytics (premium feature)
- White-label (premium feature)

**Outcome:**
- Clear MVP definition
- Faster development timeline
- Post-MVP feature roadmap created

---

### 2026-01-06: Competitive Edge Strategy

**Decision:** Triple differentiator approach (Language + Telegram + Price)

**Rationale:**
- **Bahasa Indonesia:** No direct competitor offers full localization
- **Telegram Bot:** Unique integration, fits Indonesian market preferences
- **Pricing:** 30-40% cheaper than UserJot ($29 â†’ $19-21, IDR 249K)

**Market Positioning:**
- "Feedback management untuk startup Indonesia"
- Focus on local community dan ecosystem
- Build relationships via Telegram groups

**Outcome:**
- Clear competitive advantage defined
- Marketing strategy aligned
- Pricing model established
